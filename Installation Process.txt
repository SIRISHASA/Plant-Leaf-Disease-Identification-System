Project Name: Leaf Disease
Environment Name: LeafDiseaseEnv

System Specification:
	Windows 7/8/10 
	64bit OS
	Ram: 4 GB


Create Environment: 
1. Run In Anaconda Promt
	--> conda create --name LeafDiseaseEnv python=3.6
	
	-->activate LeafDiseaseEnv


2. To install pillow
	-->pip install pillow

3. To install opencv
	-->pip install opencv-python

4. To install matplotlib
	-->pip install matplotlib

5. To install scipy
	-->pip install scipy

6. To install scikit-image(skimage)
	-->pip install scikit-image



Load The Path.
Run the project...

	-->python run.py
